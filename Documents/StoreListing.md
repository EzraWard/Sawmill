## Description (max 10,000 characters)

Sawmill is a fast, modern log file viewer built for Windows 10 and Windows 11. Tail log files in real time, search with text or regex, and highlight matches in multiple colors — all in a clean, Fluent-styled interface.

**Real-Time Log Tailing**
Watch log files update live as they're written. Sawmill streams new lines instantly with smooth virtual scrolling, keeping you at the tail of the file or letting you scroll freely.

**Powerful Search & Highlighting**
Filter lines by plain text or regular expressions. Apply multiple search terms simultaneously with different highlight colors and icons. Toggle between filtered and full-file views to see matches in context.

**Built for Large Files**
Sawmill handles files of any size — tested with files up to 47 GB. Virtual scrolling means only visible lines are rendered, so performance stays smooth regardless of file size.

**Multi-File Monitoring**
Open multiple files in tabs and monitor them side by side. Drag and drop files directly into the window to start tailing. Middle-click a tab to close it.

**Keyboard-Driven Workflow**
- Ctrl+O — Open a file
- Ctrl+T — New tab
- Ctrl+C — Copy selected lines
- Mouse wheel over tabs — Scroll through open tabs

**Customizable**
- Adjustable text zoom
- Toggle new-line highlighting
- Show or hide line numbers
- Persistent settings across sessions

**Modern Design**
Built with Windows 11 Fluent design principles — clean typography, rounded corners, smooth animations, and native window chrome integration.
